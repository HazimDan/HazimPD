Slime Destroy

This is a game that I created using the Unity application and using C# language. The purpose of this game is to destroy the slimes as they continuously spawn over time. The player is allowed to pick up the cube and cylinder to hit the slime.

Depending on the object they use to hit the slime, there will be a different animation to the slime. If they use the cube, the slime will look like it got taken damage before being destroyed and disappear whereas if they used the cylinder, the slime will melt to the ground before being destroyed and will disappear.
